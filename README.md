## 🚀 About the Project

An interactive, hyper-responsive prank dashboard built to surprise users with Aespa's media content. It features specialized browser engine filtering, custom particle mechanics, and dynamic viewport calculation logic.

### ✨ Key Features

* **Intelligent Media Automation:** Implements a platform-aware loading curtain strategy to bypass strict audio-blocking policies specifically for iOS and Safari clients.
* **Dynamic Screen Calibration:** Computes mobile-specific layout factors via runtime JavaScript initialization, preventing UI overflow caused by shifting mobile address bars.
* **Edge-to-Edge Fluidity:** Utilizes strict aspect-ratio locking containers combined with full cover scaling to eliminate visual black bar gaps on ultra-wide desktop monitors.
* **Immersive FX Layering:** Connects canvas-based confetti hardware acceleration for responsive victory celebrations upon content reveal.

## 🛠️ Tech Stack

* **Front-End core:** HTML5, CSS3 (Custom Variables, Flexbox Grid)
* **Execution Logic:** Modern Vanilla ECMAScript (Promises, User Interface Handlers)
* **Animation Core:** Canvas Confetti Engine (Asynchronous Worker-driven performance)
